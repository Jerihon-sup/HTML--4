Информативная Шпаргалка

Программирование

    Git — распределённая система контроля версий (Version Control System, VCS), которая помогает отслеживать изменения в коде, возвращаться к старым версиям и работать над проектом вместе с другими разработчиками.
    Visual Studio Code (VS Code) — кроссплатформенный текстовый редактор и интегрированная среда разработки (IDE), разработанная компанией Microsoft. Предназначен для работы с различными языками программирования и технологиями, а также для разработки кросс-платформенных десктопных приложений.

Сайты помомощники

    Tor (сокр. от The Onion Router) — свободное и открытое программное обеспечение для реализации луковой маршрутизации. Это система прокси-серверов, позволяющая устанавливать анонимное сетевое соединение, защищённое от прослушивания, и предоставляющая передачу данных в зашифрованном виде.
    IPSpeed.info — сайт, на котором доступны геолокация по IP-адресу, проверка скорости интернета, бесплатные VPN и прокси.

Полезные приложения

    Dr.Web («Доктор Веб») — семейство антивирусного программного обеспечения для различных платформ (Windows, macOS, Linux, мобильные платформы). Разрабатывается компанией «Доктор Веб».
